Project Introduction

This is SUTD Chinese Lyrical Tradition class project. In this project, we did word analysis for 787 poets from the Tang and Song Dynasties as well as Book of Odes. The result are saved as word cloud and put in the database in the website. We also made some Chinese Style pictures and displayed them on the website.

Poem Data is from: https://github.com/chinese-poetry/chinese-poetry
